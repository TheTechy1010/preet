Academic Major Project (M.Sc.)
An Android cum Web Application to estimate Nitrogen Concentration in Rice Leaf of Western Odisha
GitHub | Live
Description
  • LCC (Leaf Color Chart) is a tool made for estimating Nitrogen concentration in rice leaf as the color of the leaf corresponds to a Nitrogen level.
  • The aim of the project is to digitalize the LCC. The apps or the web app captures the rice leaf images and sends it to the server for processing and calculation. The server responds back with a HEX color code and the Nitrogen Level of the corresponding leaf with fertilizer requirments and recommendations.
  • The captured image, date, time is recorded for monitoring and evaluation purposes.
Tech Stack
  • The backend Web API is made with ASP.NET Core 3.1. The Image processing and color detection is done in using .NET and C#.
  • The frontend part of the Web version which runs on browser is made with Angular and TypeScript. The cross platform apps for Android, iOS, Desktop are made with Xamarin.Forms.
  • The SQLite database is used as frontend and the backend database. The application is hosted on the server IIS.
